# Rangaswamy Reddy Kappeta — Portfolio

Modern, responsive portfolio built with **Next.js + Tailwind + Framer Motion**.

## Highlights
- Enterprise RPA focus: **UiPath REFramework**, **Orchestrator queues**, assets, credentials
- Hybrid automation: **UI automation + REST/SOAP API integrations**
- Experience across web/Windows/Citrix, production monitoring, and migrations (Automation Anywhere → UiPath)

## Tech Stack
- Next.js (App Router)
- TailwindCSS
- Framer Motion

## Run locally
```bash
npm install
npm run dev
```

## Customize
- Update profile info in `data/site.ts`
- Update resume content in `data/resume.ts`
- Replace LinkedIn URL and project links

## Deploy
- Vercel (recommended)
- Netlify
- GitHub Pages (static export possible with adjustments)
